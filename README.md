# DevToolsMax 🛠️

> Free online developer tools — fast, clean, and privacy-friendly.

**Live site:** [devtoolsmax.me](https://devtoolsmax.me)

---

## What is DevToolsMax?

DevToolsMax is a collection of free, no-signup developer utilities that run entirely in your browser. No data is ever sent to a server — everything is processed locally on your device.

---

## Tools Available

| Tool | Description |
|------|-------------|
| JSON Formatter | Format or minify JSON instantly, with syntax validation |
| Word & Character Counter | Count words, characters, sentences and reading time |
| Base64 Encoder / Decoder | Encode or decode Base64 strings |
| URL Encoder / Decoder | Encode special characters for URLs or decode them back |
| Password Generator | Generate strong random passwords with custom options |
| UUID Generator | Generate RFC 4122 v4 UUIDs using the Web Crypto API |

## Running Locally

No setup needed. Just open `index.html` in your browser:

```bash
git clone https://github.com/UltraAce77/devtoolsmax.git
cd devtoolsmax
open index.html
```
